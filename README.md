# Saizencode(サイゼンコード)｜メニュー番号を復号化するWEBアプリ
## 必ずお読みください
## ✅はじめに
先日、友人からサイゼリヤのメニュー番号が送られてきたため、

復号化するプログラムを作成致しました。

サイゼリヤのメニュー番号をメニュー名に変換するツールです。

このツールは”非公式”です。

## 👍使い方
## ▷開発者のTwitter

友達からメニュー番号で送られたときにお使いください。
メニュー番号を入力することにより、メニュー名に変換できます。

また、メニューが変更されていたら
TwitterのDMにてお問い合わせください。
